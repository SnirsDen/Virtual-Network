# Virtual-Network
Настройка сети
Для соединения ВМ в изолированную сеть используйте Внутреннюю сеть (Internal Network):

    Для каждой ВМ зайдите в Настройки → Сеть.

    В разделе Адаптер 1:

        Выберите Тип подключения: Внутренняя сеть.

        Укажите Имя сети: (одинаковое для всех ВМ, например, my_network).
# Disable Firewall
sudo disable ufw

# Назначить IP адрес
sudo ip addr add 192.168.10.1/24 dev enp0s3

# Включить интерфейс
sudo ip link set enp0s3 up

# Проверить
ip addr show enp0s3
ping 192.168.10.1  # Должен работать (ping до себя)
